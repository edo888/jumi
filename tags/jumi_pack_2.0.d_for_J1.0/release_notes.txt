Version 2.0.d

================================
This is the next generation Jumi for Joomla! 1.0. It is based on previous versions of Jumi Module, Plugin and on Jumo component for Joomla 1.5.
All previous Jumi extensions underwent internal enhacement, improvement and mutual integration.
When upgrading from any previous version of any Jumi extension it is strictly recomended to uninstall all previous versions of any Jumi extension and install this one.
This package contains the following Jumi custom code extensions: component, content plugin, module

Installation instructions:
Install Jumi package as a standard Joomla! 1.0 component.
All extensions contained in the package are installed automatically.

Changes:
    - added new extension - Jumi component
    - module and plugin insert custom codes from a component database table too
    - module and plugin have another parameter - debug mode
    - module inserts directly written code too
    - plugin is able to process nested Jumi syntax

Demo
There are several demos there. All of them are installed automatically. Any one is accessible by any Jumi extension.
    - Component demo: there are two component demos there. The first one, Hello Jumi!, is automatically assign
the main menu item nad published. The second one, Blogspot, can be accessed through the previous demo or you can assign to it a menu item manually - by a standard Joomla! manner.
    - Module demo file: modules/mod_jumi/jumi_demo.php
    - Plugin demo file: mambots/content/jumi/jumi_demo.php
