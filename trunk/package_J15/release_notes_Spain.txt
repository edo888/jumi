Version 2.0.0
Release date: 2008/11/29
================================Esta es la primera versión de Jumi - la nueva generación. Esta basada en previas versiones del modulo Jumi, Plugin y Componente Jumi.
 Todas estas extensiones se integran perfectamente.
Cuando se actualice cualquiera de las citadas extensiones es altamente recomendado desinstalar completamente antes de instalar esta nueva versión.
Todas las extensiones contenidas en este pack se instalan automaticamente.
Este pack contiene las siguientes extensiones:
- Jumi componente
- Jumi plugin de contenido
- Jumi modulo
================================
Demo 
Se incluyen algunas demos. Todas ellas se instalan automaticamente. Cada una es accesible desde su propia extensión.
Componente demo: hay dos demos del componente en la base de datos Jumi. La primera, Hello Jumi!, es automaticamente asignada al main menu. La segunda, Blogspot, puede ser accesada a través de la primera o puedes asignarla manualmente a un item de menu - de forma estandar.
Module demo : modules/mod_jumi/jumi_demo.php
Plugin demo file: plugins/content/jumi/jumi_demo.php
================================
Nota:
- La desisntalación del componente Jumi desinstala el módulo y el plugin a la vez.
